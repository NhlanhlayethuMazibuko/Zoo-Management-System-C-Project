# Zoo Management System
A C# console application built with Three-Tier Architecture, Entity Framework Core, and SQLite.

## Tech Stack
- C# (.NET 8)
- Entity Framework Core
- SQLite

## Features
- Add, view, update, and delete parrots
- Menu-driven console interface
- Data persistence with SQLite
- Object-Oriented Programming (inheritance, polymorphism)

## Architecture
- **PL** (Presentation Layer) – Console UI and user input
- **BL** (Business Layer) – Business logic and validation
- **DAL** (Data Access Layer) – Database operations using EF Core

## How to Run
1. Open the solution in Visual Studio 2022.
2. Press CTRL + F5 to run.
3. The database will be created automatically on first run.
